# 无猜扫雷 (No-Guess Minesweeper) Ver.0.1.0

一个基于 React + Vite + TypeScript 开发的现代扫雷游戏。本项目专注于**移动端体验**、**逻辑可解性**与**智能辅助**。

## ✨ 特性说明

- **100% 逻辑可解 (无猜模式)**：内置高级求解引擎验证，保证所有布局均可通过逻辑推演确定，彻底告别“二选一”盲猜。
- **智能提示系统 💡**：不仅能指出下一步，还能识别你的插旗错误。提示后自动定位视野，助你快速突破难点。
- **现代交互体验**：
  - **丝滑操作**：针对移动端优化，支持自由拖拽、双指缩放。
  - **视觉反馈**：操作不匹配时带有震动（移动端）与红色抖动反馈。
- **多级难度**：支持初级、中级、高级、极难以及完全自定义模式。

## 🎮 基本操作

### 🖱️ 桌面端 (PC)
- **左键单击**：翻开方块。
- **右键单击**：插旗标记地雷 / 取消标记。
- **滚轮滚动**：放大或缩小棋盘。
- **鼠标拖拽**：在屏幕任何位置按下并移动，即可自由平移视野。
- **点击数字**：若周围雷已找齐，自动翻开剩余空格；若周围空格数正好等于雷数，自动插旗。

### 📱 移动端 (Mobile)
- **单指轻点**：翻开方块。
- **单指长按**：插旗标记地雷。
- **双指捏合**：缩放棋盘大小。
- **单指滑动**：自由平移视野。
- **点击数字**：同 PC 端，执行智能快捷操作。

## 🚀 部署与运行
本项目支持通过 GitHub Actions 自动部署至 GitHub Pages。
本地开发请运行：
```bash
npm install
npm run dev
```
预览构建产物：
```bash
npm run build
npm run preview
```
