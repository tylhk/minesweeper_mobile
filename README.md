# 无猜扫雷 (No-Guess Minesweeper)

一个基于 React + Vite + TypeScript 开发的现代扫雷游戏。本项目专注于**移动端体验**与**逻辑可解性**。

## ✨ 特性说明

- **100% 逻辑可解 (无猜模式)**：内置扫雷算法校验器，保证第一下点击必定大面积铺开，且所有地雷均可通过逻辑推演确定，告别运气开局。
- **现代交互体验**：
  - **移动端适配**：支持双指捏合缩放 (Pinch-to-Zoom)、单指长按插旗、全局自由拖拽。
  - **智能操作**：单击已揭开的数字可自动执行“标雷”或“开区”逻辑。
  - **视觉反馈**：操作不匹配时带有震动反馈（手机端）与红色抖动反馈。
- **多种难度挑战**：
  - 初级 / 中级 / 高级
  - **极难模式**：24x30 盘面，220 颗高密度地雷。
  - **自定义模式**：自由设定行、列与雷数。
- **明亮现代 UI**：极简清新的视觉设计，全宽磨砂质感状态栏。

## 🎮 玩法介绍

### 桌面端 (Desktop)
- **左键单击**：翻开方块 / 点击数字自动操作。
- **右键单击**：插旗标记 / 右键数字自动标记周围雷区。
- **鼠标拖拽**：在屏幕任何位置按下均可自由移动地图。

### 移动端 (Mobile)
- **单指轻点**：翻开方块 / 点击数字自动操作。
- **单指长按**：插旗标记。
- **双指操作**：捏合缩放地图。
- **单指滑动**：自由平移地图。

## 🛠️ 技术栈

- **前端**：React 19 + TypeScript
- **构建**：Vite
- **样式**：Vanilla CSS (Flexbox & Grid)
- **部署**：GitHub Pages / Vercel

## 🚀 本地开发

1. 克隆仓库：
   ```bash
   git clone https://github.com/your-username/minesweeper.git
   ```
2. 安装依赖：
   ```bash
   npm install
   ```
3. 运行项目：
   ```bash
   npm run dev
   ```
4. 局域网手机测试：
   ```bash
   npm run dev -- --host
   ```
